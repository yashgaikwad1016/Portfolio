## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/yashgaikwad1016/Portfolio.git
```

Navigate to project folder:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
